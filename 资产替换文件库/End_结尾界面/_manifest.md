## Phase B：铜镜反光→侧脸浮现（3-7s，4s 过渡）

| 文件名 | W×H | @2x | Alpha | 用途 | 动画 |
|---|---|---|---|---|---|
| `end_mirror-face.png` | 600×500 | 1200×1000 | RGBA | 封面中央的铜镜反光区——铜绿色半透明，内含模糊侧脸轮廓 | opacity 0%→60% 渐进 4s |

### 侧脸映像规格

```
尺寸:       600×500（铜镜映像的外接矩形）
位置:       居中于 album PNG 的标题区域（约 1200×800 内部的 center-top）
Alpha:      整体 0→60% opacity（不完全不透明，保持铜镜半透明感）
形状:       不规则——铜镜映像的边缘不是矩形，是逐渐淡出的铜绿色光晕
映像颜色:   铜绿 #5D7A5E (偏绿暗调) + 铜金高光 #B87333
轮廓:       模糊的低头侧脸——仅能辨认额头→鼻梁→下巴→颈部的大轮廓线
           （不是写实，是铜镜倒影的质感——断续的、斑驳的）
```

### 颜色叠加

```css
/* 引擎端混合模式 */
end_mirror-face.png 叠加在 end_album-closed.png 上:
  globalCompositeOperation: 'screen' or 'lighten'
  globalAlpha: 0 → 0.6 (4s linear)
```

## 替换步骤

1. 将所有 PNG 放入 `public/assets/epilogue/`
2. 重写 `ChapterDemoEnd.ts`（或新建 `ChapterCredits.ts`）：
   - 替换 `drawPaperBackground()` 为 `ctx.drawImage(end_bg-desk.png, 0, 0)`
   - 叠加 `end_album-closed.png`
   - 叠加 `end_vignette.png`
   - 在 B 阶段叠加 `end_mirror-face.png`，`globalAlpha` 驱动
   - 文字和按钮沿用 Canvas 2D 文字渲染
3. 在 `main.ts` 中注册为最后一个 chapter（epilogue 之后）
