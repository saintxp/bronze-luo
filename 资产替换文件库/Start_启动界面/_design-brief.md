## 画面描述

### 主体：集章册封面

一本古旧的宣纸集章册平摊在木桌上，俯视角度（轻微透视，不是正俯视）。

**封面材质：**

- 染过桐油的粗麻布包裹硬纸板，暖棕色（`#8B6F47`）
- 布面有明显经纬纹理，岁月磨损——四角有轻微起毛
- 封面中央凹陷处曾经贴过标签，现在只剩浅色矩形痕迹

**封面上元素：**

- 标题「铜声·识洛」四个字以**压凹烫金**形式在封面中央——不是印上去的，是铜模压入纸面形成的凹陷金色字痕
- 字下方一条铜色细线（约 120px 长，`#B87333`）
- 再下方小字「一座城的青铜记忆」（墨色，`#6f675d`）

**装订细节：**

- 右侧可见棉线装订——4 个装订孔，粗棉线穿过
- 棉线末端打结，有轻微散开
- 书脊处有铜绿色包角（`#5D7A5E`）——青铜时代暗示

**册子右侧：**

- 一根朱砂色（`#b64232`）丝带书签从册子中垂落
- 书签末端停在「开始游戏」按钮对应的位置
- 丝带上隐约可见编织纹理

### 环境

**桌面：**

- 深色旧木桌（`#3a3028`），能看到木纹和几处浅色磨痕
- 桌面右侧有一盏油灯（不点亮，铜质灯座+素白灯罩）
- 灯下压着一片干枯的牡丹花瓣（暗红褐色）

**光影：**

- 从左上方的窗户斜射进来的自然光
- 光线经过宣纸窗纸的柔化，没有强烈阴影
- 封面上的烫金标题在光线下微微闪光

**角落：**

- 右下角桌面边缘放着一枚铜镜（背面朝上，镜背有简单的几何纹饰）
- 铜镜旁散落几粒绿松石碎屑（`#4A9B9B`）——二里头的呼应

### 色调

```
主调：  暖棕 #8B6F47 → 画布的桐油浸染感
辅调：  铜金 #B87333 → 标题烫金、铜镜
点缀：  朱砂 #b64232 → 书签、花瓣
背景：  深木 #3a3028 → 桌面
高光：  宣纸白 #fffaf0 → 光照区域
阴影：  墨棕 #2a2723 → 暗部
```

## 禁则

- ❌ 不要出现任何「游戏 UI」感——没有按钮、没有图标、没有现代元素
- ❌ 不要出现纯白背景（`--ink-bg: #f6f1e6` 是暖米色，不是纯白）
- ❌ 不要出现任何文字之外的装饰（不要边框、不要角落花纹、不要印章在封面上——印章是游戏过程中一个个盖上去的）
- ❌ 不要出现人像或面部
- ❌ 不要把油灯画成点亮状态

## 分层建议

为了灵活性，建议分 3 层出图：

| 层 | 内容 | Alpha |
| --- | --- | --- |
| `start_bg-desk.png` | 桌面+油灯+铜镜+花瓣（静态环境） | RGB |
| `start_bg-album.png` | 集章册封面+标题+棉线书签 | RGBA（册子外缘需透明，方便叠在桌上） |
| `start_bg-shadow.png` | 册子的桌面投影+光照方向 | RGBA（半透明黑色 multiply） |

这样 HTML 层的按钮文字「开始游戏」可以叠在书签末端。
