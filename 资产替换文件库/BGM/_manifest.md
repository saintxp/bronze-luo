# BGM · 替换清单

> 11 轨背景音乐，替换当前游戏中的静默 / AudioManager 占位
> 生成工具：主用 Suno AI，精修用 Udio

---

## BGM 音频规格全局要求

| 参数 | 值 |
| --- | --- |
| 格式 | WAV 48kHz/24bit (母版) + OGG Vorbis q6 (运行版) |
| 时长 | 每轨 2:00 ~ 3:00 |
| 声道 | 立体声 |
| 响度 | -16 LUFS (integrated)，留出铜声音效动态空间 |
| 循环 | 首尾无缝衔接（尾声《合册》允许淡出） |
| 频段 | 避开 1-4kHz 人声敏感区 |
| 禁止 | 现代电子合成器、电吉他、鼓机、电子贝斯、Auto-Tune、流行和弦进行、西方管弦乐主导、歌词、人声说唱 |

## 事件总线触发名

在代码中通过 `EventBus` 触发 BGM 切换：

```ts
// 进入章节时
eventBus.emit('bgm:play', { bgmId: 'fengTi' });
eventBus.emit('bgm:play', { bgmId: 'zhiChu' });
// ...等

// 章节退出时
eventBus.emit('bgm:stop');
eventBus.emit('bgm:crossfade', { to: 'zhuHuo', duration: 1500 }); // 1.5s 渐变切换
```

## 与铜声/音效的层级关系

```
BGM（底层氛围，-20dB ~ -14dB LUFS）
  ↓
环境音（风声/水声/纸张，-24dB ~ -18dB LUFS）
  ↓
铜声音效（事件触发，-12dB ~ -6dB LUFS）
  ↓
UI 音效（翻页/盖章，-10dB ~ -6dB LUFS）
```

**关键规则**：

- BGM 不占用 1-4kHz 主频段，避免与文字阅读冲突
- 铜声触发时，BGM 做 1-2dB 闪避（可选）
- 避免 BGM 与铜声使用完全相同的乐器 solo，防止掩蔽
