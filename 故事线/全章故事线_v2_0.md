# 三、教学关 L1-L3（6帧·独立预算·线稿风格）

> 在正式章节前插入。极简美术（线稿或单色），每个 2 帧，不占正式帧预算。
> 按 L1→L2→L3 顺序线性推进，不可跳过。

### L1 · 接触 —— 「拖拽对位」

| 项目 | 内容 |
|------|------|
| 教学点 | 拖拽一个画面格到另一个画面格上，两者对齐后触发事件 |
| 机制 | 空门框 + 散落门板。拖拽门板到门框 → 对齐吸附 → 门关闭 |
| 帧数 | 2帧（空门框+门板 / 关门完成） |
| 对应正式章 | 纸成(帘→水槽)、托(刀背→掌纹)、烬(烟雾→塔檐) |

### L2 · 透视 —— 「嵌套/窗口」

| 项目 | 内容 |
|------|------|
| 教学点 | 画面中的洞是通往另一层的窗口。平移画面让洞对准下一层。 |
| 机制 | 圆形洞口→透过洞口看到模糊颜色→平移对准→物体清晰→画面放大进入第二层 |
| 帧数 | 2帧（第一层洞口 / 嵌套进入第二层） |
| 对应正式章 | 灰页(裂缝嵌套) |

### L3 · 合力 —— 「角度与共振」

| 项目 | 内容 |
|------|------|
| 教学点 | 旋转一个格去匹配另一个格的"方向"，找到共振角 |
| 机制 | 两个齿轮格。一个固定旋转方向，一个可旋转。齿啮合→两齿轮同时转动 |
| 帧数 | 2帧（两齿轮分离 / 啮合转动） |
| 对应正式章 | 瘦骨(双力共振) |
