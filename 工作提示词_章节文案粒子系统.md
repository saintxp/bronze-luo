# 工作提示词：铜声·识洛 章节文案粒子系统

> 实施前请先阅读项目 CLAUDE.md 和 .pi/skills/tongzheng-dev/SKILL.md

## 任务

为《铜声·识洛》游戏创建 ParticleColophon 组件，并在全部 29 处（登录页 + 28 处章节完成点）集成竖排粒子文案动画。

## 项目上下文

- 项目路径: E:\projects\铜声·识洛
- 技术栈: HTML5 Canvas 2D + TypeScript 5.x + Vite
- Canvas 尺寸: 1280x720（src/utils/constants.ts）
- 设计系统: InkView 水墨风 + 青铜器色板
- 现有 UI 组件: DefinitionPopup.ts, StampEffect.ts, TutorialOverlay.ts
- UI Canvas 由 CanvasManager 管理多层画布

## 实施阶段

### Phase 0: 粒子原型验证

- 路径: src/tests/particle-colophon-prototype.html
- 独立 HTML 页面，验证粒子汇聚/吹散效果在竖排中文下的表现
- 离屏 Canvas 渲染竖排中文(64px bold)，getImageData() 采样像素点
- 粒子数 ≤5000，≥30fps
- 逐列汇聚: 右列先成形，左列依次，每列间隔 0.3s
- 逐列吹散: 左列先散，右列依次

### Phase 1: 组件开发 + 首章集成

- src/ui/ParticleColophon.ts
- 集成到 ChapterErlitou.ts 作为试点
- 同时完成登录页粒子题记

### Phase 2: 全量覆盖(29 处)

P0 章优先 → P1 章 → 过渡/暗页 → 教学关 → 尾声

## 重要约束

1. 不要修改 DefinitionPopup 和 StampEffect
2. 粒子总数 ≤5000
3. 竖排从右往左,多列时右列先汇聚
4. 字号 22-26px,离屏渲染用 64px bold
5. 墨水风克制: 粒子如墨迹渗入宣纸、被风吹散
6. clear() 时释放离屏 Canvas 和粒子数组
