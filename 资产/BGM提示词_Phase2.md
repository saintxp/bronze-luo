# 铜声·识洛 — BGM 提示词（Phase 2 专用）

> **范围**：开始界面 + 教学关 L1-L3 + 序章 + 壹·二里头 + 灰页 + 贰·周王城  
> **已确认**：采用方案 A，从全游戏 11 轨中裁出 6 轨。

## 00 开始界面 · 《封题》

- **时长**：2:00
- **乐器**：古琴散音、埙长音、纸张低频震动、远处编钟残响
- **情绪**：神秘、温暖、邀请，像古册封面在黑暗中呼吸
- **循环**：长呼吸式铺底，任意位置可切入

**中文 Suno 提示词**：

```
中国水墨风格游戏开始界面背景音乐，

主要乐器为古琴散音和埙长音，加入极淡的纸张低频震动与远处编钟残响，

可无缝循环，无歌词，无人声，
像在邀请玩家打开一本沉睡的册子
```

