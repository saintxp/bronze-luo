# 铜声·识洛 — Phase 2 图片资产提示词

> **用途：** 替换当前 Canvas 2D 占位渲染为 Seedream 4K 正式资产
> **生成工具：** 即梦 Seedream 4K（静态图）
> **视频工具：** 即梦 Seedance 1.5 Pro（视频片段）
> **色板基准：** InkView 水墨设计系统 + 青铜色扩展（详见 CLAUDE.md）
> **总资产数：** ~48 张静态图 + 4 段视频

---

## 通用风格指令（每张图前加这段）

```
水墨风插画，暖宣纸底纹理，仿古书册质感，文物细节严谨，哑光色调，无明显光源阴影，纯色背景无环境光散射，半透明纸层叠加效果，历史感沉淀。构图饱满居中，留白克制。
```

## 一、教学关 L1-L3（6张 · 线稿风格）

> **尺寸：** 1920×1080 (全景) 或 800×600 (元素)

## 四、灰页：悬置（9张 · 全灰渐变→玄色）

> **总风格：** 全灰色调从 `#8B8070`（灰）渐变为 `#1A1A18`（玄色），暗金色边缘光 `#C8A65A`。
> **约束：** 除边缘光外无其他暖色。无青铜声对应——灰页是沉默的悬置。
> **关键视觉手法：** 每层的裂缝形状不同，越往后裂缝越细、灰越深。

### 0G-01 灰页整页（背景 — 1920×1080）
```
画面：全页均匀烟灰色 (#8B8070)，页面整体平坦如茧，纸面微有纤维纹理（极淡）。页面上没有任何图案、文字或装饰。纯粹的灰色——无限平静，像是时间暂停在这一页。无裂缝，无瑕疵。
```

### 0G-02 裂缝第1层（谜题元素 — 1920×1080）
```
画面：在 0G-01 的全灰底上，出现第一道裂缝。裂缝呈树枝状分叉，从画面左上延伸至中心。裂缝边缘染暗金色 (#C8A65A，极淡，透明度约 15%)。裂口内部透出下层灰色（稍浅 #9B9080）。裂缝宽度约 2-3px。
```

### 0G-03 裂缝第2层（谜题元素 — 1920×1080）
```
画面：灰底色略加深至 (#8B8070 → #7A7060)，裂缝增加第2道分叉。裂缝边缘暗金色加深至 25% 透明度。裂缝比第1层更细（约 1.5-2px）。裂缝分布呈枝状+细碎星纹。裂缝内透出更浅灰色 (#A09080)。
```

### 0G-04 裂缝第3层（谜题元素 — 1920×1080）
```
画面：灰底色继续加深至 (#6A6050)，裂缝增加第3组。暗金色边缘光 35% 透明度。裂缝呈蛛网状分布，某些交叉点形成微小孔洞。裂缝宽度 1-1.5px。透过最大裂缝可见下一层深灰色 (#5A5040)。
```

### 0G-05 裂缝第4层（谜题元素 — 1920×1080）
```
画面：灰底近玄色 (#302828)，裂缝群密集如冰裂。暗金色边缘光 45%。裂缝极细（约 1px），但在某些交点扩大为小缺口（暴露下层文字）。文字通过缺口隐约可见但不完整——竖笔画的局部，无法辨认内容。
```

### 0G-06 裂缝第5层→暗页显现（谜题元素 — 1920×1080）
```
画面：底色近纯玄色 (#1A1A18)，一道主裂缝从画面中央贯穿，裂口宽度约 3-5px，暗金色边缘光 (#C8A65A) 明亮（60%透明度）。裂口内透出暖光背景，暗示隐藏内容即将揭晓。
```

### 0G-07 观星者笔记油纸（背景 — 1200×800）
```
画面：一张旧油纸（半透明，浅棕色 #e8dcc8），上面有针孔星图——用针刺出的小孔排列成北斗七星和二十八宿部分星群。星图右下角有竖排小字「平子先生授」，字迹为汉隶风格，淡墨书写。油纸边缘有烧痕和虫蛀小孔。纸张纤维可见。
```

### 0G-08 「下一季，玄鸟归来」褪灰露出（关键帧 — 1920×1080）
```
画面：玄色 (#1A1A18) 底色正在褪去，从中浮现一组文字「下一季，玄鸟归来」。文字为淡金色 (#C8A65A)，字体篆书。文字呈现方式不是写在上面，而是从底色中剥露出来——像是底层白纸原本就有这行字，只是灰色正从此处退去。画面下方隐约有一只玄鸟（燕子）剪影。
```

### 0G-09 龙尾轮廓→蜷曲→绳三帧序列（VFX关键帧×3 — 1920×1080）
```
提示词：三帧序列——①龙尾灰色轮廓在玄色背景上浮现，轮廓模糊。②轮廓向内蜷曲，尾部线条收窄。③蜷曲后的轮廓压缩为绳结状索纹。全灰度色调，无暖色。类似旧照片褪色效果。三帧间过渡平滑。
```
