# 铜声·识洛 — BGM 循环点说明（方案 A · 10 轨）

> 用于音频编辑时标记循环点，以及引擎 `BGMManager` 实现无缝播放。

---

## 通用标记约定

- **Intro Bar(s)**：非循环引子，首次进入时播放。
- **Loop Start**：循环段开始位置（样本点或小节数）。
- **Loop End**：循环段结束位置，播放完此点后跳回 Loop Start。
- **Outro Bar(s)**：非循环尾声，章节结束时播放（可选）。
- **BPM**： beats per minute。
- **Time Signature**：除《悬置》外均为 4/4。
- **Key**：五声调式主音。

## 引擎实现建议

```typescript
interface BGMTrack {
  id: string;
  src: string;
  bpm: number;
  loopStart: number;  // seconds
  loopEnd: number;    // seconds
  introEnd?: number;  // seconds, optional
  outroStart?: number; // seconds, optional
}

// 示例
const tracks: BGMTrack[] = [
  {
    id: 'tutorial',
    src: 'bgm_01_zhiChu_tutorialTouch.ogg',
    bpm: 72,
    loopStart: 0,
    loopEnd: 32.0,
  },
  {
    id: 'grey',
    src: 'bgm_04_xuanZhi_greySuspended.ogg',
    bpm: 0,
    loopStart: 10.0,
    loopEnd: 175.0,
  },
];
```

## 音频编辑备注

1. **对速**：生成后先在 DAW 里按 BPM 对齐网格，确认 AI 是否稳定。
2. **Crossfade**：灰页、烬余等持续音景需 3-5 秒 crossfade 循环。
3. **响度匹配**：所有循环段标准化到 -16 LUFS，避免切换时跳变。
