## 二、Suno 提示词格式

Suno 的音效/音效生成功能更适合用**英文文本描述 + 风格标签**。以下每段可直接复制到 Suno 的「Custom Mode → Sound Effects」或类似入口。

通用后缀：

```
minimal reverb, dry close-mic, no music, no melody, single event, cinematic
```

## 四、Suno 生成参数建议

| 参数 | 建议值 |
| ------ | -------- |
| 时长 | 按上表，留 10-20% 余量便于裁剪 |
| 风格标签 | `sound effect`, `cinematic`, `ancient bronze`, `no music`, `no melody` |
| 避免 | 旋律、节奏、鼓点、现代电子音色 |
| 输出格式 | WAV 或 FLAC，保留动态范围 |

## 五、后处理 checklist

1. **裁剪静默头尾**：每个音效保留干净起振和衰减尾音。
2. **标准化音量**：所有铜声峰值约 -6dBFS，S04/S11/S14 等短音效可略高。
3. **循环标记**：S12 心跳需标记无缝循环点。
4. **格式转换**：最终输出为 `.ogg` 或 `.mp3` 供 Web Audio API 使用，但保留 WAV 母版。
5. **命名一致**：使用 `sXX_拼音_英文描述.ext` 格式，与 `AssetManifest.ts` 中 ID 对应。
