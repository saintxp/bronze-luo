18|## 盖章点一：字立（熹平石经）
19|
20|### 1. 按纸贴碑
21|
22|画面从白发纸纤维过渡——纸浆蒸干后，画面显出东汉洛阳太学门前。
23|
24|四十六块石碑立在太学门外。碑面上密密麻麻刻满了经文。隶书。一尺高。
25|
26|碑前站着一个人——背对玩家。穿太学生常服。他正在往碑面上贴一张纸。
27|
28|玩家触碰那人贴的纸——手指按上去的一刻，纸上的墨迹从纸面浮起。不是浮到空中——是在纸与碑面之间的那一层空气里，墨字脱离了纸，浮在了半空中。
29|
30|碑上的石刻经文和浮空的墨字一一对应——一笔不差。
31|
32|玩家按住纸→纸贴在碑上→浮空墨字归位。纸上的经文和碑上的经文重叠在一起，分不出哪个是纸上的、哪个是碑上的。
33|
34|### 2. 焦尾琴（V26 人物标志强化·蔡邕）
35|
36|画面外传来一声木头爆裂的声音。不是普通的柴裂——这声音有自己的音高。
37|
38|贴碑的那人（刻经人）停住了。他侧过头——画面切到不远处的一间炉灶。
39|
40|炉灶里一段桐木正在燃烧。火焰舔着桐木的表面——桐木在火里发出一种类似于弦乐器共振的声音。不是「噼啪」——是木头在唱。
41|
42|他冲过去，徒手从火里抢出那段还没烧尽的桐木。袖子烧着了——他用另一只手拍灭袖子上的火，眼睛没离开木头。
43|
44|他叩了叩烧焦的桐木——木头回了一声。像是琴已经做完了，他只是把它从火里取出来。
45|
46|这段焦桐后来被他斫成了焦尾琴。
47|
48|### 3. 焦尾琴靠碑·琴弦自震
49|
50|焦尾琴靠在碑座旁。碑上的字刚刻完最后一划——不是「邕」字，是经文正文的最后一个字。
51|
52|弦嗡的一声自己响了。
53|
54|不是风吹的。是石头的共振。碑身最后一凿的振动通过石头→地面→琴箱→弦。焦尾琴替碑唱了最后一个音。
55|
56|### 4. 盖章·释义
57|
58|玩家在碑前按下纸张→盖章落下。释义弹出：「**字立**」。
59|

62|## 盖章点二：地听（张衡地动仪）
63|
64|### 1. 针孔星图（V26 人物标志强化·张衡）
65|
66|灰页中发现的油纸星图被另一个人拿在手里——就是那个刻经人。
67|
68|他把油纸举到灯下——针孔被灯光穿过，每一孔对应一颗星，星光投影在碑面上。二十八宿在石碑上排列成一个完整的星环。
69|
70|【跨章呼应】——灰页底部「平子先生授」+ 东汉「星移。碑在。」两人通过一张油纸跨越了120年。
71|
72|### 2. 地动仪龙首
73|
74|画面切到灵台。地动仪立在中央——八条龙首各朝一方。
75|
76|龙首口中的铜丸悬在龙舌上。不是固定的——是轻轻托着的，像龙舌在舔舐一颗铜珠子。
77|
78|玩家触碰其中一条龙首。不触时不动。
79|
80|远处一声闷响——是地震的余波传到了洛阳。龙首突然转动——不是整条龙转，是龙舌往上一顶。
81|
82|铜丸从龙口坠落——**「嗒」**。
83|
84|【铜声谱系·第三声：嗒（坠落）】
85|与编钟「嗡」形成对比：钟是持续的振动，丸是单一的质量坠落。
86|
87|### 3. 铜丸找到龙首
88|
89|铜丸不是直接落进蟾蜍嘴里——铜丸在坠落的瞬间暂停了一帧，然后方向微调，精确地落进了下方蟾蜍的嘴里。
90|
91|这一帧暂停是铜对二里头记忆的回响——铜记得自己被铸成液体的那一刻。
92|
93|丸入蟾口。释义弹出：「**地听**」。

97|## 盖章点三：纸成（蔡伦造纸）
98|
99|### 1. 端帘
100|
101|画面切到一间造纸作坊。一帘湿纸浆从水槽里端起来——玩家需要帮工匠「端帘」：按住竹帘边缘向下压，把纸浆均匀压在帘面上。
102|
103|这不是一次按到位的——需要边按边移位，让纸浆均匀分布。
104|
105|### 2. 纸浆里的白发
106|
107|纸浆里嵌着一根微灰色的白发——比周王城纸浆中的纯白稍微灰了一点。但那根白发的卷曲方向和嵌在纤维里的角度，和周王城老子白发一模一样。
108|
109|【白发三色·周→东汉】
110|玩家可以看到——不是同一张纸，是同一根白发。它从周的纸浆里被抄进了东汉的纸浆。纸是新的，头发是老的。
111|
112|### 3. 盖章·释义
113|
114|纸帘出水，纸面蒸干，释义弹出：「**纸成**」。

118|## 盖章点四：托（党锢·望门投止）
119|
120|### 纸态切换：正常→拒水
121|
122|纸面开始泛潮——不是水，是纸的纤维在收缩。纸面从暖黄变成冷白的半透明质感。
123|
124|### 1. 刀背敲手
125|
126|画面：夜色。太学门外。一个太学生在往洛阳城门跑。
127|
128|一只手从背后抓住他的肩膀——是来搜捕的兵。
129|
130|刀举起来了——不是刀刃，是刀背。刀背敲在太学生握门环的手背上。不是砍下来——是敲。
131|
132|手松开了。门没推开。太学生被拖走了。
133|
134|他不是那个刻经人——他是另一个。刻经人已经逃走了。这个没逃掉的，把名字留在了门环上——门环被他的手掌捂热过。
135|
136|### 2. 野生牡丹（V26 东汉补牡丹）
137|
逃走的刻经人跑上了邙山。回望洛阳——城墙上的火把一排一排亮起来。他脚下的山坡上有一个浅浅的凹陷——雨水积在里面，反着火光，像一只眼睛在流泪。**北邙第一环·泪眼首现。**

他低头——脚边有一株野生牡丹。粉白色单瓣，就是土里长出来的一棵草花。他在杂草堆里差点踩到它——绕了一步。
141|
142|【三大IP·牡丹跨章线】
143|东汉：邙山草花（没人知道它是牡丹）
144|曹魏：花苞标记在地图上，脉动一次
145|隋唐：花苞绽开，全城盛放
146|尾声：应天门楼下绿化带——子子孙孙
147|
148|### 3. 盖章·释义
149|
150|门环上留下捂热的手印——手印中央裂开一道血纹。释义弹出：「**托**」。不是"扛"，不是"担"。是**托**——把某个比命重的东西轻轻地托了一下，然后被敲碎了。

## 🧩 V31 可玩性集成

东汉共4个盖章点，V31为全部补入谜题。

### 拼接谜题 🧩 #4：字立·按哪里

| 项目 | 内容 |
|------|------|
| **来源** | V30 #5（修改后） |
| **类型** | 拖拽对位 |
| **机制** | 纸上墨字浮空后——找纸上哪一位置按压能让浮空字与碑文精确叠合。三个可选位，仅一处正确 |
| **渐进反馈** | 靠近正确位置→浮空字与碑文对齐、墨色变深。错误→偏移/模糊 |
| **帧数** | **4帧**（纸格/3可选位各1帧/归位成功） |
| **优先级** | P1 |

### 拼接谜题 🧩 #5：地听八龙首方向判定 🏆

| 项目 | 内容 |
|------|------|
| **来源** | V30 #6（全作最强方案） |
| **类型** | 方向判定（DirectionPuzzle） |
| **机制** | 地震余波→画面出现震源方向提示（地面裂痕→建筑倾斜→水面波纹指向）。玩家从八条龙首中选出朝向震源的那一条 |
| **渐进反馈** | ①细微裂痕 ②建筑倾斜角 ③高亮正确龙首。选错→龙首轻颤不坠丸 |
| **帧数** | **5帧**（8龙首全景/3层方向提示各1帧/铜丸坠落帧） |
| **优先级** | P0 |

### 拼接谜题 🧩 #6：水帘对位

| 项目 | 内容 |
|------|------|
| **来源** | V31 |
| **类型** | 拖拽对位（四档吸附） |
| **机制** | 水槽格(俯视纸浆纤维)+帘格(空竹帘)+水面格(倒影)。拖拽帘格到水槽格——帘纹须与纤维走向垂直(90°)。0°/30°/60°/90°四档 |
| **帧数** | **6帧**（水槽/帘/水面/90°成功/60°部分/30°微附着） |
| **优先级** | P1 |

|------|------|
